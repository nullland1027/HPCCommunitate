# HPCCommunicate

## File Structure
```
client.py: For the socket communication practice, 
server.py: For the socket communication practice
submitnode.py: Advanced socket communication practice, can be used to submit jobs to the server and blocked until the job is done.


# Final SMMCC files
task1.py
task2.py: 2 tasks for the final project smmcc.
cmd.py: The command line interface for the final project smmcc.
controller.py: The controller for the final project smmcc. Like the client
mpi.py: The manual mpi implementation. a set of functions.
node.py: The node for the final project smmcc. Like the server.
```
